# Rock Paper Scissors 🎮

A simple and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript.

## ✨ Features
- Play against a computer with random choices
- Real-time score tracking
- Basic animations and feedback
- Fully responsive UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)
